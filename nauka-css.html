Style CSS.

No właśnie, co to takiego ten cały CSS? Najkrócej mówiąc jest to język służący do formatowania wyglądu elementów HTML, które po wyświetleniu w oknie przeglądarki internetowej tworzą wygląd strony internetowej. Ponadto sam skrót CSS oznacza "Kaskadowy Arkusz Stylów" jest to plik z rozszerzeniem .css, w którym znajdują się pewne reguły CSS, czy jak kto woli pewne style CSS, które określają wygląd różnych elementów HTML znajdujących się na danej stronie internetowej.
